# Projet intensif knowledge documents FRONT

## Description du projet
Ce projet constitue le frontend du site web. Ce dernier a pour but de permettre à un utilisateur de trouver une/des 
réponses ciblées adaptées à sa recherche, à partir d'un ensemble de documents et de connaissances partagés. L'interface 
est décomposée en deux parties : les fonctionnalités utilisateur et les fonctionnalités administrateur. Cette seconde 
partie permet la gestion des ressources. 

## Mise en place
1. Cloner le projet et l'ouvrir
2. Vérifier que la version 24.12.0 de Node.js est utilisée
2. Installer les dépendances en tapant la commande _npm install_ dans un terminal (en se plaçant à la base du projet)
3. S'assurer que le backend du projet est bien lancé
4. Créer un fichier _.env_ à la racine du projet contenant la ligne suivante à personnaliser :
    ```VITE_BACKEND_API_URL=<URL du backend à insérer ici>```

## Lancement
Taper la commande _npm run dev_ dans un terminal (toujours en se plaçant à la base du projet).
Si la fenêtre du navigateur ne s'ouvre pas automatiquement, se rendre sur l'URL indiquée dans le terminal suite à la 
commande précédente.
